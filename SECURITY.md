# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Yes    |

## Reporting a vulnerability

BookTracker CLI is a local-only tool — it has no network access, no server
component, and stores all data in a plain JSON file on your own machine.

If you discover a security issue (e.g. path traversal in the `--store` flag,
or unsafe deserialization), please open a GitHub issue tagged `security`.

For genuinely sensitive issues, email the maintainer directly via the contact
on [waqarjaved.com](https://waqarjaved.com/).
